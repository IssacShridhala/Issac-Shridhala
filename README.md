# Coding Test – Calculator Program Using Class

## Test Question
Create a small calculator that performs the following operations using a **class**:

- Addition  
- Subtraction  
- Multiplication  
- Division  

### Calculator Inputs
- `a` → double  
- `b` → double  
- `type of operation` → string  
(Example: "add", "sub", "mul", "div")

**The Code is Written in C++ programming language**
